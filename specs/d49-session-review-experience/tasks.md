## D49 验收清单

- [ ] 每位符合资格的玩家每场只有一条综合游后感。
- [ ] 星级为 1–5，文字最多 900 字，页面显示实时字数。
- [ ] 不存在角色/剧本分栏、标签或分项评价。
- [ ] 最多 9 张照片且不支持视频。
- [ ] 相册选择和手机上传只能选一种；没有可选相册图片时不显示相册入口。
- [ ] 手机上传自动进入本场相册；新评价只保存相册照片引用。
- [ ] 历史评价图片继续可读，只改文字或星级不会被清空。
- [ ] 发布后可在评价分享页分享给好友、群聊和朋友圈。
- [ ] 公开页不泄漏车局成员、精确时间、相册标签或私有审核内容。
- [ ] 界面匹配用户确认的第三版并延续现有设计系统。
- [ ] D15、D18、D31、D45、D46 回归通过。
- [ ] `npm run check` 与 `npm run build:mp-weixin` 通过。
