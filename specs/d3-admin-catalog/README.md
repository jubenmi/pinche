# D3 Admin Catalog Spec

更新日期：2026-06-11

本目录是D3阶段的开发规格源。D3目标是把“系统管理员录入店家/剧本、审核新增资料申请”做成小程序内可用能力，并让车头在建车前能搜索管理员录入的店家和剧本，找不到资料时提交申请。

执行状态：已完成，详见 [tasks.md](./tasks.md) 的验证记录。

## Spec文件

- [requirements.md](./requirements.md)：D3需求与验收条款。
- [design.md](./design.md)：管理员资料页、API、种子数据和验收设计。
- [tasks.md](./tasks.md)：D3执行任务与完成状态。

## D3冻结目标

```text
系统管理员入口仅 system_admin 可见
管理员能搜索、创建、编辑、下架店家
管理员能搜索、创建、编辑、下架剧本
剧本支持默认座位模板
管理员能审核新增资料申请
车头能搜索 active 店家/剧本
车头找不到资料时能提交新增申请
项目有基础种子数据
```

D3不实现完整D4建车流程、座位编辑器、补贴配平组件或正式发布车体验。
