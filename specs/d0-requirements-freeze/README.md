## D0冻结结论

MVP只验证一条主链路：

```text
系统管理员录入店家/剧本
  -> 车头选择店家/剧本建车
  -> 车头配置座位和补贴
  -> 车头发布并分享
  -> 玩家从分享页申请上车
  -> 车头审核申请
  -> 车头锁座并标记押金状态
```

D0后新增想法默认进入后续版本池，除非它阻塞上面这条主链路的可用性或微信小程序合规性。
