# D9 Review Materials

更新日期：2026-06-13

## 小程序信息草稿

- 名称：剧本迷·拼车
- 简介：剧本杀发车与座位申请工具，支持车头建车、玩家申请和车头审核。
- 服务类目：按微信后台可选类目选择与“剧本杀/桌游/休闲娱乐/工具”最贴近且真实匹配的类目。

## 审核账号

开发态 mock 仅用于本地调试：

```text
管理员：dev-admin-openid
普通用户：任意 dev-* code
```

正式提审不能依赖 mock login。正式提审需要准备真实微信体验成员，并确保首批管理员 OpenID 已配置到生产环境。

## 审核说明草稿一

本小程序用于剧本杀发车拼车管理。用户可浏览发车详情，选择座位提交申请；车头可创建发车、设置座位实付价、分享车详情、审核申请并记录线下锁位状态。小程序不提供现金红包、返现、抽奖、平台支付、平台代收押金或分享奖励。分享仅用于用户自愿扩散发车信息和来源统计，不兑换任何权益。

## 审核说明草稿二：体验路径

审核人员可按以下路径体验主流程：

1. 打开小程序并完成微信登录。
2. 进入首页浏览可用店铺和剧本信息。
3. 进入建车流程创建发车。
4. 打开发车详情并查看座位信息。
5. 从分享页或车详情页提交上车申请。
6. 使用车头身份进入管理页审核申请。
7. 通过申请并记录线下锁位状态。

如需要管理员资料录入路径，请使用已配置为管理员 OpenID 的体验微信账号进入管理员资料页。

## 隐私保护指引字段

- 微信登录标识：用于识别用户身份。
- 用户填写的联系方式：用于车头审核和联系上车，不公开展示。
- 报名备注：用于车头审核，不公开展示。
- 分享来源参数：用于统计发车来源，不兑换权益。
