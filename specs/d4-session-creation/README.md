# D4 Spec: 车头建车流程

更新日期：2026-06-12

D4目标是把“车头选择管理员资料 -> 配置时间、DM/NPC、押金和座位 -> 校验补贴配平 -> 发布车”做成可验收闭环。

## Spec文件

- [Requirements](./requirements.md)
- [Design](./design.md)
- [Tasks](./tasks.md)

## 阶段出口

- 车头能从小程序建车页完成一辆草稿车创建。
- 车头能用剧本默认模板或空白模板配置座位。
- 座位实付价和补贴合计实时计算。
- 补贴合计不为0、实付价小于0、缺少店/本/时间/座位时不能发布。
- 发布成功后进入车详情页。
- 微信开发者工具中可验证建车页无业务错误。
