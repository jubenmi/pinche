# 首页日历操作图标设计

日期：2026-07-10

## 目标

让首页日历上方三个辅助操作的图标语义更清楚，并统一为圆角线性 SVG 风格。

## 设计

- 管理入口：将当前工具箱图标替换为齿轮图标，明确表达“设置/管理”。
- 归位：使用左向尖头与回转弧线组成的“返回/复位”图标，表达把时间轴视图重新定位到今天；不再使用容易被理解为定位的准星图标。
- 日期：将“日期”或已选择日期文字替换为日历图标，点击行为仍为打开日期选择器。
- 三个图标沿用现有按钮配色。管理入口使用浅色图标；归位和日期使用主题绿色图标。
- 归位和日期统一使用同一种按钮组件、同一个 `24 × 24` SVG 视觉边界、线宽和图标尺寸，消除普通 `view` 与 `t-button` 内部基线不同造成的高低错位。
- 两个按钮保留当前尺寸、边框、间距和点击区域，不调整日历布局。
- 纯图标按钮添加明确的 `aria-label`：管理、归位到今天、选择日期。

## 资源与实现边界

- 使用项目内本地 SVG，不新增图标依赖或网络资源。
- SVG 采用一致的圆角端点、连接方式和视觉线宽。
- 不修改 `scrollToToday`、日期选择器、筛选或其他业务逻辑。

## 验证

- 运行小程序静态检查和构建，确认 SVG 资源被正确打包。
- 检查三个按钮仍触发原有操作，图标在浅色和绿色按钮上均清晰可见。
- 确认小屏下两个日历工具按钮不溢出、不改变现有布局，并且图标中心与视觉基线对齐。
